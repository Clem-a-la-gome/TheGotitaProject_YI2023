About

This app can be used by:

  • Engineers: for improving water quality and network performance.
  • Policymakers: for conceiving better public health policies.
  • Operators: they can be alerted if there is a serious problem related to the water network.

This is the first version of the GotitaApp and will be modified. Please contact us if you need more information and/or if you have any suggestions for future versions.
